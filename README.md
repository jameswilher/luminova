# Luminova Lighting Decor

This is a single-file GitHub Pages website for a lighting decor project.

## Important

Upload `index.html` to the root of your GitHub repository.

Do not upload only the ZIP file. GitHub Pages needs the actual `index.html` file in the repository.

## Publish on GitHub Pages

1. Open your GitHub repository.
2. Upload `index.html`.
3. Go to Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: main.
6. Folder: /root.
7. Save.
8. Wait 1 to 5 minutes, then refresh your website link.
